\section{Citations and Bibliographies}

This thesis uses the \texttt{natbib} bibliography system with the \texttt{plainnat} style.
All references are managed through three \texttt{.bib} files:
\begin{itemize}
    \item \texttt{references.bib} --- primary reference database exported from Zotero
    \item \texttt{references\_non\_zotero.bib} --- manually curated entries not in Zotero
    \item \texttt{IEEEabrv.bib} --- IEEE abbreviation database for journal names
\end{itemize}

All citations in the text follow the numbered reference style \texttt{[1]}, \texttt{[2]}, \ldots,
using \verb|\cite{key}|. The bibliography is rendered at the end of this document.
Each entry includes the Digital Object Identifier (DOI) where available to ensure
persistent link resolution.
